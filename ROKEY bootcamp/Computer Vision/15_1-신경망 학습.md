### 수치 미분
$$\dfrac{df(x)}{dx}=\lim_{h\to 0} \frac{f(x + h) - f(x-h)}{2h} $$
![[Pasted image 20240903120957.png|500]]
수학에서는 h를 0으로 수렴 시키는데 PC에서는 불가능하니 h값을 매우 작게 설정.
```python
def numerical_diff(f, x):
    h = 10e-4
    return (f(x + h) - f(x - h)) / (2 * h)
```
