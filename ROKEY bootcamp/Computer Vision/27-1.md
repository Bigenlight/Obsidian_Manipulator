#### Thread를 씀
qt는 무조건 event loop이라 thread를 써야 다른 기능 동시 사용 가능.
```python
import threading

th = threading.Thread(target=run)
```

참고로 파이썬에서는 전역 변수를 함수 안에서 사용하려고  할때 변수 앞에 무조건 global를 붙여야 함.
