인스턴스를 만들고(app)
.exec_()로 무한 루프 돌림.

 ```python
import sys
from PyQt5.QtWidgets import QApplication, QMainWindow, QWidget, QLabel

app = QApplication(sys.argv)

window = QMainWindow()
window.setWindowTitle('Hello, World! - mainwindow')
window.show()
  
sys.exit(app.exec_())
```
